# Requirements

| REQ-ID | Description                                                                                           | Type           | Priority |
| ------ | ----------------------------------------------------------------------------------------------------- | -------------- | -------- |
| 001    | The system allows creation of predefined lighting modes                                               | Functional     | Must     |
| 002    | A lighting mode can be applied to single room                                                         | Functional     | Must     |
| 003    | The system allows configuring lighting settings once and synchronizing them across all selected rooms | Functional     | Must     |
| 004    | The system allows grouping of similar room types                                                      | Functional     | Should   |
| 005    | Guests can view the current lighting mode of their room                                               | Functional     | Must     |
| 006    | Allows customization of an existing lighting mode                                                     | Functional     | Must     |
| 007    | The system is usable by non-technical hotel staff                                                     | Non-functional | Must     |
| 008    | System acknowledges abnormal situations such as fire alarms and changes lighting mode accordingly     | functional     | Must     |
| 009    | The system ensures consistent lighting configurations across all synced rooms                         | Non-functional | Must     |
| 010    | Access to system configuration is blocked from unwanted parties                                       | Non-functional | Must     |
| 011    | System automatically shuts light when room is not booked to save energy                               | functional     | Should   |
