# Prototype Scope

## 1. Prototype Description

The prototype is a simple tool on the terminal that demonstrates how hotel lighting can be configured once and synchronized across multiple rooms. It allows it's user to define lighting modes and apply them to individual rooms or groups of rooms. The prototype is intended as a demonstration for potential customers on the ease of use of RoomLight service. It proves that centralized configuration and synchronization is easily achieved and valuable.

---

## 2. Demonstrated Requirements

- REQ-001: Shows creation of predefined lighting modes  
- REQ-002: Shows applying a mode to a single room  
- REQ-003: Demonstrated synchronizing between rooms  
- REQ-006: Allows customization of an existing lighting mode
- REQ-007: System is usable by non-technical hotel staff

These requirements were selected because they are the most essential in showing the value of RoomLight for potential customers
